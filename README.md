# GuestInsight AI - Homestay Review Sentiment Classifier

## Project Overview

GuestInsight AI is a full-stack web application that helps homestay owners analyze guest reviews. Users can submit reviews, and the system stores them in MongoDB while displaying sentiment, theme, and a suggested management response.

---

## Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

---

## Features

- Add guest reviews
- View all reviews
- Update existing reviews
- Delete reviews
- RESTful API integration
- MongoDB Atlas cloud database
- Responsive user interface
- Dark mode support
